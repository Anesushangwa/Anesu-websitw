The process of searching for the errors and finding them is called debugging.
Debugging JavaScript code is difficult because the browser doesnâ€™t show any

different approaches to tackle the problem.

The console.log() methodworking with the Browser Console.
You can easily open the console by right-clicking on the page, then selecting the
Inspect tab. Or you can use the shortcuts Ctrl+Shift+I or F12 for the same purpose.

In JavaScript, the debugger keyword stops the execution and if available, calls the debugging function.
The keyword works the same way as setting a breakpoint. In the absence of debugging, the debugger statement has no effect. With
the debugger turned on, this code stops executing when it encounters a debugger statement.
